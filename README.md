# vsrKiCad
秋月電子通商で売られている部品のKiCadライブラリです。無保証、自己責任でご使用ください。

「環境変数の設定」パネルで、名前「VSRKICADLIB」にリポジトリのパスを指定すると、3Dデータのパスがセットできます。

パスの例：

Name   |  Path
-------|-------
VSRKICADLIB  | /Users/yourname/GitHUB/vsrKiCad
